🚀 AI CORE v5.7 | VOID ENGINE

AI CORE — это легковесный и мощный бенчмарк для стресс-тестирования видеокарты, написанный на Python. Утилита использует OpenCL для проведения тяжелых математических вычислений непосредственно на чипе вашего GPU, позволяя оценить его реальную вычислительную мощность.



✨ Основные возможности

Hardware Stress Test: Использование ядер OpenCL для цикличных математических операций (50,000 итераций на юнит).



Real-time Scoring: Динамическое отображение производительности в PTS.



Global Ranking: Встроенная таблица сравнения вашего результата с эталонными картами серии RTX 30/40/50.



Hardware Info: Автоматическое определение модели вашего CPU и GPU через системные вызовы.



Modern UI: Темный футуристичный интерфейс на базе библиотеки customtkinter.



🛠 Технологический стек

Язык: Python 3.14+



Библиотеки: customtkinter, pyopencl, numpy



Движок вычислений: OpenCL (с фолбеком на DWM/DirectX отрисовку при отсутствии драйверов).



🚀 Запуск и установка

Из исходного кода:

Убедитесь, что у вас установлен Python.



Установите необходимые зависимости:



Bash

pip install customtkinter pyopencl numpy

Запустите скрипт:



Bash

python aimark.py

Готовый файл (.exe):

Вы можете скачать скомпилированную версию в разделе \[Releases] на этом GitHub.
